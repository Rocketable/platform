# Platform

- `rocketclaw`: bridges Slack and Discord connectors to RocketCode. Run `rocketclaw help` for setup, doctor, OpenAI login, and session inspection commands.
- `rocketcode`: experimental interactive CLI used to test RocketCode behavior in a workspace.
- `rocketloop`: non-interactive RocketCode loop for a goal supplied as arguments or stdin. Supports `--script`, `--max-loops`, and `--script-output-limit`.
